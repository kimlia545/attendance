# attendance_check
opencv project (web)

# Process

* Mask Detect
* Eye Blink Detect
* Face Recog and Save attendance record (Name, Time)
* Register new person 

# Tech/framework used
- Python
- Flask
- OpenCV
- MySQL(HeidiSQL)

# Models 
- faceNet
- maskNet
- blink_model
- face_recognition dlib
